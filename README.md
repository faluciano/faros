# Faros (WIP)

Web app to track visited lighthouses around the world with a map to display visited points

# Functionality

- [ ] Lighthouse map
- [ ] Login/Register
- [ ] Visited Lighthouse map
- [ ] Friends
- [ ] Combined friend/personal map
- [ ] Saved Lighthouses
- [ ] Lighthouse visit upload

# Stack

- Go (Backend)
- React (Frontend)
- Clerk (Auth)
- Postgress (DB)
- React Leaflet (Maps)
